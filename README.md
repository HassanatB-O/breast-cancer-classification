# Breast Cancer Classification Model

Overview

This project implements a Breast Cancer Classification Model using machine learning to predict whether a tumor is malignant or benign based on diagnostic features.

The model is trained using the Breast Cancer dataset from scikit-learn and demonstrates a complete ML workflow: data preprocessing, model training, evaluation, and prediction.

<h2>Technologies & Libraries Used</h2>

<li>Python</li>

<li>Pandas – data manipulation</li>

<li>NumPy – numerical operations</li>

<li>Matplotlib – exploratory data analysis & visualization</li>

<li>Scikit-learn – model training & evaluation</li>

The model achieves strong classification performance on the test set, demonstrating the effectiveness of feature scaling and probability-based classification.
